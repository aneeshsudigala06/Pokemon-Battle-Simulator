# Pokémon Battle Simulator
## Project Description
* Developed a ML pipeline to simulate and predict the outcomes of Pokémon battles using data of 800+ Pokémon from Generations 1 – 7.
* Engineered an engine to simulate and calculate performance scores by integrating statistics (HP, Attack, etc.) with multipliers for type effectiveness.
* Trained a Logistic Regression model using Scikit-Learn to analyze differentials (Speed, Legendary Status, etc.) to determine probabilities of winning.
## Techologies Used
* CSV
* Python
  * Pandas
  * Scikit-Learn
